    • Hashy

    Get a handle on your #hash.

    Project maintained by ethagnawl Hosted on GitHub Pages — Theme by mattgraham

    Hashy is a utility function which allows for CRUD operations on `window.location.hash`.

    Get

    window.location.hash = 'key=value';
    hashy(); // '#key=value'

    Set

    window.location.hash = 'key=value';
    hashy('new_key=new_value');
    hashy(); // '#new_key=new_value'

    Destroy

    hashy('key=value');
    hashy(false);
    hashy(); // ''

    Set k,v pair

    hashy('key', 'value');
    hashy(); // #key=value

    Update k,v pair

    hashy('key', 'value');
    hashy('key', 'new_value');
    hashy(); // #key=new_value

    Append k,v pair

    hashy('key', 'value');
    hashy('key_1', 'value_1');
    hashy(); // #key=value&key_1=value_1

    Set multiple k,v pairs

    hashy(['key', 'value'], ['key_1', 'value_1']);
    hashy(); // #key=value&key_1=value_1

    Update multiple k,v pairs

    hashy(['key', 'value'], ['key_1', 'value_1']);
    hashy(['key', 'new_value'], ['key_1', 'new_value_1']);
    hashy(); // #key=new_value&key_1=new_value_1

    Append multiple k,v pairs

    hashy('key', 'value');
    hashy(['key_1', 'value_1'], ['key_2', 'value_2']);
    hashy(); // #key=value&key_1=value_1&key_2=value_2
